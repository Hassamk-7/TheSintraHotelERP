6.	SendReservation (Mandatory)
Description: This method sends the reservation.
Important Notes:
●	In the production environment; when all prerequisites met; If property has legacy bookings before activation of the integration, these past reservations are expected to be sent via this method with setting multiple reservations in Reservations[] array object.
●	System will not respond with error or warning messages if the reservation dates are past dates, and will recognize them as past reservations in the central reservation system.
Prerequisites:
●	Hotels have to be mapped.
●	Rooms have to be mapped.
●	Rates have to be mapped.
●	Rates for the target rates should be sent.
●	Allotment should be set and available.
